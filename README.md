# Home Services Marketplace

A minimalist MERN stack On-Demand Home Services application.

## Quick Start

1.  **Install Dependencies**

    ```bash
    cd home-service-marketplace
    cd server && npm install
    cd ../client && npm install
    ```

2.  **Run Application**
    - **Backend**: `cd server && npm start` (Port 5000)
    - **Frontend**: `cd client && npm run dev` (Port 5173)

## Functionality

- **URL**: [http://localhost:5173](http://localhost:5173)
- **Roles**:
  - **Kirtan**: Customer (Request services)
  - **Bhavesh**: Provider (Accept/Complete jobs)
  - **Admin**: Cancel bookings
- **Database**: Connected to MongoDB Atlas.
